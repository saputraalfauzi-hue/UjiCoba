header, nav, main, section, article, footer

CSS Dasar

CSS memberikan gaya dan tata letak pada website Anda.

Selektor Dasar

element, class, id, universal, grouping

Box Model

margin, border, padding, width, height

Typography

font-family, size, weight, color, line-height

Warna & Background

color, background-color, background-image

Layout Dasar

display, position, float, clear, z-index

Responsive Basics

media queries, viewport, percentage units

JavaScript membuat website Anda interaktif dan dinamis.

Variabel & Tipe Data

let, const, var, string, number, boolean, array, object

Operator

Fungsi

function declaration, expression, arrow functions

Array Methods

map, filter, reduce, forEach, find, some, every

DOM Manipulation

getElementById, querySelector, innerHTML, classList

localStorage, sessionStorage, cookies

pattern matching, validation, string manipulation
