JavaScript Dasar

Pelajari JavaScript untuk membuat website interaktif dan dinamis.

  • Variabel dan tipe data
  • Fungsi dan kontrol alur
  • DOM Manipulation
  • Event Handling
  • Fetch API dan AJAX

Contoh: Manipulasi DOM

// Seleksi elemen
const button = document.querySelector('#myButton');
const output = document.querySelector('#output');

// Tambahkan event listener
button.addEventListener('click', function() {
    output.textContent = 'Tombol diklik!';
    output.classList.add('active');
});

// Manipulasi elemen
function updateContent() {
    const element = document.getElementById('content');
    element.innerHTML = 'Konten diperbarui';
}

JavaScript Lanjutan

Konsep JavaScript tingkat lanjut untuk pengembangan yang lebih kompleks.

  • ES6+ Features
  • Asynchronous JavaScript
  • Object-Oriented Programming
  • Functional Programming
  • Modules dan Bundlers

Contoh: Async/Await

// Fungsi asynchronous dengan async/await
async function fetchData() {
    try {
        const response = await fetch('https://api.example.com/data');
        const data = await response.json();
        return data;
    } catch (error) {
        console.error('Error fetching data:', error);
    }
}

// Penggunaan
fetchData().then(data => {
    console.log('Data diterima:', data);
});

// Arrow function dan array methods
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map(num => num * 2);
const evens = numbers.filter(num => num % 2 === 0);

Framework & Tools

Pelajari framework modern dan tools untuk pengembangan web efisien.

  • Version Control dengan Git
  • React.js atau Vue.js
  • Package Managers (npm, yarn)
  • Build Tools (Webpack, Vite)
  • Testing (Jest, Cypress)

Contoh: React Component

import React, { useState } from 'react';

function Counter() {
    const [count, setCount] = useState(0);
    
    const increment = () => setCount(count + 1);
    const decrement = () => setCount(count - 1);
    
    return (
        <div className="counter">
            <h2>Counter: {count}</h2>
            <button onClick={increment}>+</button>
            <button onClick={decrement}>-</button>
        </div>
    );
}

export default Counter;

Optimasi & Deployment

  • Web Performance Optimization
  • SEO Best Practices
  • Security Considerations
  • Deployment Strategies
  • Monitoring dan Analytics

Contoh: Web Performance Tips

// Lazy loading images
<img src="placeholder.jpg" data-src="image.jpg" class="lazy" alt="...">

<script>
document.addEventListener("DOMContentLoaded", function() {
    const lazyImages = document.querySelectorAll('img.lazy');
    
    const imageObserver = new IntersectionObserver((entries) => {
        entries.forEach(entry => {
            if (entry.isIntersecting) {
                const img = entry.target;
                img.src = img.dataset.src;
                img.classList.remove('lazy');
                imageObserver.unobserve(img);
            }
        });
    });
    
    lazyImages.forEach(img => imageObserver.observe(img));
});
</script>

Glosarium Web Development

Referensi lengkap kode HTML, CSS, JavaScript beserta penjelasan

HTML Elements & Attributes

<!DOCTYPE html>

HTML

Deklarasi tipe dokumen, menentukan versi HTML (HTML5).

Kegunaan: Harus ada di awal setiap dokumen HTML.

<html lang="id">

HTML

Elemen root dari dokumen HTML dengan atribut bahasa.

Kegunaan: Menentukan bahasa halaman untuk aksesibilitas dan SEO.

<head>

HTML

Container untuk metadata tentang dokumen HTML.

Kegunaan: Berisi title, styles, scripts, dan meta informasi.

<meta charset="UTF-8">

HTML

Menentukan encoding karakter untuk dokumen HTML.

Kegunaan: Memastikan karakter ditampilkan dengan benar.

<title>

HTML

Menentukan judul halaman web yang muncul di tab browser.

Kegunaan: Penting untuk SEO dan pengalaman pengguna.

<body>

HTML

Berisi konten yang terlihat dari halaman web.

Kegunaan: Tempat semua konten halaman (teks, gambar, dll).

CSS Properties & Selectors

display: flex;

CSS

Mengatur container sebagai flexbox untuk layout fleksibel.

Kegunaan: Membuat layout responsif dengan penjajaran dan distribusi ruang yang mudah.

grid-template-columns

CSS

Mendefinisikan kolom dalam grid layout.

Kegunaan: Membuat layout kompleks dengan baris dan kolom.

@media (max-width: 768px)

CSS

Media query untuk styles pada perangkat dengan lebar maksimum 768px.

Kegunaan: Membuat desain responsif untuk tablet dan mobile.

position: absolute;

CSS

Memposisikan elemen relatif terhadap elemen parent terdekat yang diposisikan.

Kegunaan: Untuk penempatan presisi elemen di halaman.

z-index: 10;

CSS

Mengontrol tumpukan vertikal elemen yang diposisikan.

Kegunaan: Menentukan elemen mana yang muncul di depan atau belakang.

transition: all 0.3s ease;

CSS

Menerapkan transisi halus pada perubahan properti CSS.

Kegunaan: Membuat animasi halus untuk interaksi pengguna.

JavaScript Methods & Concepts

document.querySelector()

JavaScript

Mengembalikan elemen pertama yang cocok dengan selector CSS.

Kegunaan: Seleksi elemen DOM untuk manipulasi.

addEventListener()

JavaScript

Mendaftarkan event handler untuk elemen tertentu.

Kegunaan: Menangani interaksi pengguna seperti klik, input, dll.

fetch()

JavaScript

API modern untuk melakukan HTTP requests.

Kegunaan: Mengambil data dari server secara asinkron.

map()

JavaScript

Membuat array baru dengan hasil pemanggilan fungsi pada setiap elemen.

Kegunaan: Transformasi data dalam array.

filter()

JavaScript

Membuat array baru dengan elemen yang lolos tes fungsi.

Kegunaan: Filter data dalam array berdasarkan kondisi.

async/await

JavaScript

Sintaks untuk menangani operasi asinkron dengan cara yang sinkron.

Kegunaan: Menulis kode asinkron yang lebih mudah dibaca dan dipelihara.

Responsive Design

Kegunaan: Meningkatkan pengalaman pengguna di desktop, tablet, dan mobile.

Umum

Kegunaan: Meningkatkan ranking website di hasil pencarian organik.
